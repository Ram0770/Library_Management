# Library Management

Simple front-end project for a Library Management demo.

- **Author:** Ram0770
- **Contact:** ramson0770

## Files

- `index.html` — main page
- `index.css` — styles
- `index.js` — client-side behavior

## Usage

Open `index.html` in your browser to run the app locally.

Alternatively, serve with a simple HTTP server from the project folder:

```powershell
python -m http.server 8000
# then open http://localhost:8000
```

## Notes

This repository contains only the static frontend files. You can add a backend or expand features as needed.
